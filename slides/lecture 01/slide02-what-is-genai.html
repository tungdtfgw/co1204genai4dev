What is Generative AI?

  • Generative Artificial Intelligence: is a branch of AI that focuses on creating new, unique, and meaningful content, rather than merely analyzing or classifying existing data.
    • Creative Capability: Gen AI can generate text, images, audio, video, source code, and many other types of data that previously only humans could create.
    • Learning from Data: Generative AI models learn patterns, structures, and relationships from large amounts of existing data to create new content that is similar but not a copy.
  • Distinction from Discriminative AI:
    • Discriminative AI: Focuses on classification or prediction based on input data (e.g., image recognition, spam email classification).
    • Generative AI: Focuses on creating new data (e.g., generating images from descriptions, writing code from requirements).
