Add MCP to VS Code Copilot

  • Enable MCP in Settings: VS Code Copilot has experimental support for MCP.
  • You can enable it by editing your settings.json file.
  • Configuration: Add the github.copilot.experimental.mcp object to your settings.
  • Inside, provide the path to your MCP server's executable command.
settings.json JSON 
{
  "github.copilot.experimental.mcp": {
    "my-local-server": {
      "command": "/path/to/your/mcp/server/executable"
    }
  }
}
