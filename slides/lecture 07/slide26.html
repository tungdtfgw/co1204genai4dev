File Manager: Defining Tools (`@server.list_tools`)

  • This function returns a list of `Tool` objects.
  • Each `Tool` tells the AI client what it can do, what inputs it needs, and what the purpose of the tool is.
file_manager_server.py Python 
# From file_manager_server.py

@server.list_tools()
async def list_tools() -> list[Tool]:
    return [
        Tool(
            name="read_file",
            description="Read file content",
            inputSchema={
                "type": "object",
                "properties": { "path": { "type": "string" } },
                "required": ["path"]
            }
        ),
        Tool(
            name="write_file",
            description="Write content to file",
            inputSchema={
                "type": "object",
                "properties": {
                    "path": { "type": "string" },
                    "content": { "type": "string" }
                },
                "required": ["path", "content"]
            }
        ),
        # ... other tools like list_files ...
    ]
