Bước 2: Load và Prepare IMDB Data

  • Load Dataset: IMDB movie reviews đã được cleaned và preprocessed
  • Label Mapping: Convert text labels ('positive', 'negative') thành numerical (1, 0)
  • Data Split: 80% training data, 20% testing data cho evaluation
  • Stratified Split: Giữ nguyên tỉ lệ positive/negative trong cả train và test
  • Ready for Training: Data structure hoàn chỉnh cho fine-tuning process
  • Data Validation: Check data distribution và quality trước khi training
data_preparation.py Python 
# Load IMDB dataset
df = pd.read_csv('IMDB_cleaned.csv')
print(f"Total samples: {len(df):,}")

# Convert sentiment to numerical labels
df['label'] = df['sentiment'].map({
    'positive': 1, 
    'negative': 0
})

# Split data: 80% train, 20% test
train_texts, test_texts, train_labels, test_labels = train_test_split(
    df['cleaned_review'].tolist(),
    df['label'].tolist(),
    test_size=0.2,
    random_state=42,
    stratify=df['label']
)

print(f"Training: {len(train_texts):,} samples")
print(f"Testing: {len(test_texts):,} samples")
