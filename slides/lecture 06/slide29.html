Bước 3: Load Pre-trained DistilBERT Model

  • DistilBERT Selection: Version nhẹ của BERT với 66M parameters, fast training
  • Tokenizer Loading: Convert text thành tokens mà model hiểu được
  • Classification Head: Configure cho 2 classes binary classification
  • Label Mapping: Định nghĩa ID và nhãn tương ứng (0=NEGATIVE, 1=POSITIVE)
  • Device Transfer: Chuyển model lên GPU/MPS để tăng tốctraining
  • Model Statistics: Check số parameters và memory usage
model_loading.py Python 
# Choose pre-trained model
MODEL_NAME = 'distilbert-base-uncased'

# Load tokenizer
tokenizer = AutoTokenizer.from_pretrained(MODEL_NAME)
print("Tokenizer loaded")

# Load model for sequence classification
model = AutoModelForSequenceClassification.from_pretrained(
    MODEL_NAME,
    num_labels=2,
    id2label={0: "NEGATIVE", 1: "POSITIVE"},
    label2id={"NEGATIVE": 0, "POSITIVE": 1}
)

# Move to device
model = model.to(device)
print(f"Model moved to: {device}")

# Model statistics
total_params = sum(p.numel() for p in model.parameters())
print(f"Parameters: {total_params:,}")
