Setup & Initialization

  1. Create your API key in AI Studio and export it as GEMINI_API_KEY.
  2. Install the Gen AI SDK (pip install -q -U google-genai).
  3. Import the library and construct a client.
  4. For single-turn interactions use generate_content; for conversations create a chat session.
initialization.py Python 
from google import genai
client = genai.Client()
