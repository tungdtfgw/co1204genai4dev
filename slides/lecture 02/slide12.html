When to Use Stateless Interactions

  • One-off questions where no context needs to be preserved.
  • Generating summaries, translations or other transformations of a single input.
  • Situations where you want full control over the prompt without hidden state.
  • Avoid stateless calls for multi-turn conversations because the model won’t remember earlier messages.
