Getting Your API Key

  • To use the Gemini API you need an API key. Create one for free in Google AI Studio.
  • Once created, set it as an environment variable GEMINI_API_KEY (or GOOGLE_API_KEY) so the client library can pick it up automatically.
  • It’s recommended to set only one of these environment variables. If both are set, GOOGLE_API_KEY takes precedence.
bash Shell 
export GEMINI_API_KEY=<YOUR_API_KEY>
