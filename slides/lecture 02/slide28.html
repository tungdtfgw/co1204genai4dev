Best Practices & Cautions

  • Never expose your API key; store it securely and rotate it regularly.
  • Follow Google’s usage policies and safety guidelines when deploying applications.
  • Set environment variables (GEMINI_API_KEY or GOOGLE_API_KEY) instead of hard-coding secrets.
  • Monitor latency and cost; disable thinking when speed is more important than depth.
  • Stay within model context limits and plan for summarization or context compression for long sessions.
