Make Your First Request

  • After setting your API key, create a client and send a request to generate content.
  • The client automatically picks up the key from GEMINI_API_KEY.
  • Alternatively you can pass the key explicitly when creating the client.
request_example.py Python 
from google import genai

client = genai.Client()

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents="Explain how AI works in a few words"
)
print(response.text)
