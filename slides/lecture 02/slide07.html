Install the Gen AI SDK

  • The official client libraries simplify authentication and API calls.
  • Install the Python library with pip.
  • For JavaScript/TypeScript developers, install with npm.
bash Shell 
# Python
pip install -q -U google-genai

# JavaScript/TypeScript
npm install @google/genai
