System Instructions

  • System instructions guide the model’s behaviour before it processes user prompts. They help you define a persona, tone or set of rules.
  • To provide a system instruction, pass a GenerateContentConfig with a system_instruction field when calling generate_content.
  • System instructions are processed only once at the start of a generation, so you don’t need to include them in every prompt.
