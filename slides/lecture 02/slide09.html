Thinking Budget & Model Config

  • The thinking feature of Gemini 2.5 series improves response quality but can increase response time and token usage.
  • You can disable or control thinking by setting a thinking budget in the generation config.
  • Disabling thinking can reduce cost and latency. Note that thinking is enabled by default on Gemini 2.5 Flash and cannot be disabled on Gemini 2.5 Pro.
config_example.py Python 
from google import genai
from google.genai import types

client = genai.Client()
response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents="Explain how AI works in a few words",
    config=types.GenerateContentConfig(
        thinking_config=types.ThinkingConfig(thinking_budget=0)
    )
)
print(response.text)
