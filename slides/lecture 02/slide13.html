Stateful Conversations

  • For multi-turn dialogues you can create a chat session. The chat object stores a history of messages so the model can respond with context.
  • Each call to send_message appends a new turn to the chat and returns the model’s response.
  • Chat sessions enable more natural conversations, follow-up questions and continuity across turns.
