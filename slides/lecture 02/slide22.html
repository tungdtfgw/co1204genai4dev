Create a Client & Chat Session

  • Use the client to make stateless calls or to create a chat.
  • The stateless call doesn’t retain memory.
  • The chat session stores the conversation history automatically.
client_chat_creation.py Python 
from google import genai
client = genai.Client()
# Stateless call
resp = client.models.generate_content(model='gemini-2.5-flash', contents='Hello!')
print(resp.text)

# Stateful chat
chat = client.chats.create(model='gemini-2.0-flash')
chat_resp = chat.send_message(message='Hello!')
print(chat_resp.text)
