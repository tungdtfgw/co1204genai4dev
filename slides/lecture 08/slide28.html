Triage & Guardrail Integration

  • Central router: triage agent decides which tutor should answer each question.
  • Guardrail enforcement: InputGuardrail plugs in the asynchronous homework check.
  • Handoff list: math and history tutors are registered for automatic delegation.

# tut02_3agents.py — triage agent with guardrail
triage_agent = Agent(
    name="Triage Agent",
    instructions="You determine which agent to use based on the user's homework question",
    handoffs=[history_tutor_agent, math_tutor_agent],
    input_guardrails=[
        InputGuardrail(guardrail_function=homework_guardrail),
    ],
)
