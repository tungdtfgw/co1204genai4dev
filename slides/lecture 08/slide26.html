Asynchronous Guardrail Flow

  • Async evaluation: guardrail agent runs with await Runner.run(...).
  • Structured result: final_output_as(HomeworkOutput) enforces typed data.
  • Tripwire control: flips to True whenever the input is not homework.

# tut02_3agents.py - guardrail coroutine
async def homework_guardrail(ctx, _agent, input_data):
    # Run the guardrail agent to analyze the input
    result = await Runner.run(
        guardrail_agent,
        input_data,
        context=ctx.context,
    )

    # Extract the structured output
    final_output = result.final_output_as(HomeworkOutput)

    # Guardrail triggers when the question is not homework
    return GuardrailFunctionOutput(
        output_info=final_output,
        tripwire_triggered=not final_output.is_homework,
    )
