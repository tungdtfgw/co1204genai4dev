Summarizing a PDF (Python)

  • Fetch a remote PDF and encode as bytes: Download PDF content and prepare it for processing
  • Use inline data in generate_content: Send the PDF data directly to Gemini for analysis
pdf_summarization.py Python 
from google import genai
from google.genai import types
import httpx

client = genai.Client()
doc_url = "https://example.com/report.pdf"
# Download PDF bytes
doc_data = httpx.get(doc_url).content

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=[
        types.Part.from_bytes(data=doc_data, mime_type='application/pdf'),
        "Summarize this document"
    ]
)
print(response.text)
