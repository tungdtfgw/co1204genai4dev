Image Captioning Example

  • Provide an inline image and ask for a caption: Load image data and send it to Gemini for analysis
  • Reading the caption result: Process the response and display the generated caption
image_captioning.py Python 
from google import genai
from google.genai import types

client = genai.Client()
with open('picture.jpg', 'rb') as f:
    image_bytes = f.read()

contents = [
    types.Part.from_bytes(data=image_bytes, mime_type='image/jpeg'),
    'Caption this image.'
]
response = client.models.generate_content(
    model='gemini-2.5-flash',
    contents=contents
)
print(response.text)
