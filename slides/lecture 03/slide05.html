Text‑to‑Image Example (Python)

  • Create a client and request an image: Initialize the Gemini client and generate content using the image model
  • Saving the generated image: Process the response and save the generated image to a file
text_to_image.py Python 
from google import genai
from PIL import Image
from io import BytesIO

client = genai.Client()
prompt = "A futuristic city skyline at sunrise with flying cars"

response = client.models.generate_content(
    model="gemini-2.5-flash-image-preview",
    contents=[prompt],
)

for part in response.candidates[0].content.parts:
    if part.inline_data:
        image = Image.open(BytesIO(part.inline_data.data))
        image.save("city_sunrise.png")
        print("Image saved.")
