Audio Summarization Example

  • Upload an MP3 file and ask for a description: Use the Files API to upload audio and request analysis
  • Display the model's response: Process and display the generated summary or transcript
audio_summarization.py Python 
from google import genai

client = genai.Client()
# Upload audio file
myfile = client.files.upload(file="lecture.mp3")
# Ask the model to describe the clip
response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents=["Summarize the main points", myfile]
)
print(response.text)
