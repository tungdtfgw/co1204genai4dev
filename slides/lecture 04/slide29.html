Output Caching Overview

  • Output caching is a method of storing and retrieving entire prompt-response pairs.
    • This approach differs from input caching which only stores the model's internal state.
    • It helps reduce latency to nearly zero and maximize cost savings when there's a cache hit.
  • Output caching supports two matching methods.
    • Exact matching requires the prompt to be exactly the same.
    • Semantic matching allows cache hits even when prompts are not completely identical.
