Gemini - Explicit Caching

  • Explicit caching allows developers to control cache behavior and TTL.
  • It provides more control over cache invalidation and specific caching requirements.
  • Developers can specify cache TTL and cache keys.
  • This is useful for applications that need to manage cache invalidation.
gemini_explicit_caching.py Python 
# Create explicit cache with Gemini 2.0 Flash
cache = client.caches.create(
    model='gemini-2.0-flash',
    config=types.CreateCachedContentConfig(
        display_name='PDF document cache',
        system_instruction=(
            'You are an expert document analyzer. Answer questions '
            'based only on the content of the provided PDF document. '
        ),
        contents=[pdf_file],
        ttl="3600s",  # Cache for 1 hour
    )
)

print(f"Cache created successfully: {cache.name}")
print(f"Cache TTL: {cache.ttl}")
